# Changelog

## [0.1.0] - 2026-06-01

### Initial release — Zed port

- Dark and light variants for Zed
- Botanical ink palette: hibiscus, sage, river moss, slate sky, thistle, ochre
- WCAG 2.1 compliant (main text AAA, all accents AA or above)
- Warm paper backgrounds with depth hierarchy (dark well pattern)
- Borders à la Flexoki — quasi-invisible, alpha-based
- Ports included: Zed, VS Code, Kitty, Alacritty, Ghostty

### Planned

- Neovim port
- iTerm2 port
- Marketplace publication (VS Code, Zed extensions)
